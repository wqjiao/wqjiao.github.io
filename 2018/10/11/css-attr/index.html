空格处理

一、空格规则

HTML 代码的空格通常会被浏览器忽略。


上面是一行 HTML 代码，文字的前部、内部和后部各有两个空格。
浏览器的输出结果如下:
   

<p>hello
world</p>


hello world


<p>hello<br>world</p>


    html:
        <p>  hellohellohello hello
        world
        </p>
    style:
        p {
            width: 100px;
            background: red;
        }
    

    p {
        white-space: nowrap;
    }
    

    p {
        white-space: pre;
    }
    

    p {
        white-space: pre-wrap;
    }
    

    p {
        white-space: pre-line;
    }
    ```
    除了文本内部的换行符没有转成空格，其他都与normal的处理规则一致。这对于诗歌类型的文本很有用。

弹性布局 flex

align-content 父元素属性 - 纵向

- flex-start
- flex-end
- center
- space-between
- space-around
- stretch

align-self 子元素属性 - 纵向

- flex-start  - 居上对齐
- flex-end  - 居下对齐
- center  - 居中对齐
- baseline  - 基线对齐
- stretch  - 填充容器（默认属性，仍保持自身的最大最小宽度）
