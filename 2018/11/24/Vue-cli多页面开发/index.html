现如今比较流行的三大前端框架，vue、react、angular，其中优势是实现前端页面的单页面应用，但是也会有需求是需要多页面开发的，这篇文章总结了 vue-cli 实现多页面配置的步骤…

搭建vue-cli项目

  • 1、全局安装vue-cli

    npm install -g vue-cli

  • 2、搭建新项目

    vue init webpack vue-multiple-pages

  • 3、进入项目

    cd vue-multiple-pages

  • 4、初始化安装依赖

    npm install

  • 5、执行项目

    npm run dev

  • 6、项目打包

    npm run build
