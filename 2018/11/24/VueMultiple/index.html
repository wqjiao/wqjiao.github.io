Vue-cli工程的多页面开发配置步骤

搭建vue-cli项目

  • 1、全局安装vue-cli

    npm install -g vue-cli

  • 2、搭建新项目

    vue init webpack vue-multiple-pages

  • 3、进入项目

    cd vue-multiple-pages

  • 4、初始化安装依赖

    npm install

  • 5、执行项目

    npm run dev

  • 6、项目打包

    npm run build
